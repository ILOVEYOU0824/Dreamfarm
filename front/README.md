# 특수교육 활동 기록 시스템

학생들의 활동 기록을 관리하고 AI로 분석하는 시스템입니다.

## 프로젝트 구조

```
.
├── front/          # 프론트엔드 (React + Vite)
└── back/           # 백엔드 (Node.js + Express)
```

## 기술 스택

### 프론트엔드
- React 19
- Vite
- Recharts (차트 라이브러리)

### 백엔드
- Node.js
- Express
- Supabase (데이터베이스)
- Google Gemini AI (AI 분석)

## 배포

- 프론트엔드: Netlify
- 백엔드: Render

자세한 배포 방법은 `배포_가이드_간단버전.md`를 참고하세요.
